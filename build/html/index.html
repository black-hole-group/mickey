Methods to read and visualize PLUTO’s output.

class pluto.Pluto(i=0)¶

Class that defines data objects imported from PLUTO.

The object’s attributes are:

  • x1,x2,x3
  • v1,v2,v3
  • pressure p
  • rho
  • n1,n2,n3
  • bx1, bx2, bx3

To read the simulation output for frame 10:

>>> import pluto
>>> p=pluto.Pluto(10)

Plots density field:

>>> p.snap()
contour_newgrid(n=200, xlim=None, rhocut=None)¶

Creates a new object with variables in cartesian coordinates. Useful if the object was created by PLUTO in polar coordinates.

Parameters:
  • n – New number of elements n^2.
  • xlim – Boundary for the plot and the grid
  • rhocut – Variable used if you want to put a lower limit to the contours
contours(N, lim, plot_flag='y')¶

Function for contour plotting. It recieves some parameters to foward to respective functions. :param N: Size of grid :param lim: is the plot limit :param plot_flag: control the plot of the density map

snap(n=20, lim=10, rhomax=2, stream='n', mag='n', var=None, hor=None)¶

Creates snapshot of 2D simulation generated in any coordinates.

Parameters:
  • n – Number of uniform divisions in x and y for the quiver plot
  • lim – The limits which the graph will be plotted (from -lim to lim)
  • var – variable to be plotted. If not specified, assumes rho
  • hor – plots circle at inner boundary radius with radius=hor. If None, no circle
 
>>> p=pluto.Pluto(10)
>>> p.snap(10,p.p)
